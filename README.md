# Algoritmos-FATEC
Algoritmos e Técnicas de Programação - FATEC Ipiranga
Este repositório contém os exercícios e projetos práticos desenvolvidos durante a disciplina de Algoritmos e Técnicas de Programação do curso de Big Data para Negócios.

O objetivo aqui é consolidar os fundamentos da lógica de programação e a resolução de problemas utilizando a linguagem Python.

 Sobre a Disciplina
A disciplina aborda conceitos essenciais para a construção de algoritmos, incluindo:

Estruturas de decisão e repetição;

Manipulação de variáveis e tipos de dados;

Listas, tuplas e dicionários;

Funções e modularização;

Lógica aplicada à análise de dados.

🛠️ Tecnologias Utilizadas
Linguagem: Python 3

Ambiente de Desenvolvimento: Google Colab

 Estrutura do Repositório
Os arquivos estão organizados por temas ou listas de exercícios:

lista_01_introducao.ipynb: Exercícios básicos de entrada e saída.

lista_02_estruturas_decisao.ipynb: Exercícios com if, elif e else.

projeto_final/: Pasta contendo o desafio principal da matéria.
